こちらはポートフォリオ用に作成したWEBアプリです。

機能

当アプリにてご利用できる主な機能を紹介します。

サーバ・チャンネル

サーバ・チャンネル

ユーザが所属するサーバ、サーバ内のチャットスペースとなるチャンネルを生成する機能を実装しました。（ 全デバイス対応 ）

サーバ追加は、ログイン後「Add Server」より、自身で作成するまたは既存サーバを検索し参加する方法で実行できます。チャンネル追加は、サーバ管理者のみ、サーバ内サイドバーの「+」より実行できます。 サーバ・チャンネルともに、「」より設定情報の変更が行えます。

今後はサーバ管理者の権限委託機能、ban機能、パスワード設定機能 etc.を追加し、セキュアなサーバ管理実現を考えています。

ビデオチャット

ビデオチャット

P2Pを用いた、ビデオ・音声共有機能、ミュート機能、ディスプレイ表示切替機能を実装しました。（ PCのみ対応 ）

ビデオチャットは以下のフローで開始できます。
1. サイドバーの「」押下
（ サイドバーに電話帳が表示 ）
2. 電話帳の任意のユーザ押下
（ 通信が始まり、発信側のサイドバーに発信画面、受信側のサイドバーに受信画面が表示 ）
3. 受信側が受信画面の「」を押下

ビデオチャットは、複数人での通話機能・顔検出してエフェクトをつける機能など、試してみたいことが多くあり、手を加えればもっとおもしろくなりそうです。

メッセージ

メッセージ

ファイル添付できるメッセージ送信機能、メッセージ編集機能、添付画像のスライド表示機能を実装しました。（ 全デバイス対応 ）

テキスト入力またはファイル添付後、「Enter」でメッセージが送信されます。 「+」よりファイル添付、「」より添付取消が可能です。 テキスト編集は「」より、ファイル削除は「」より行えます。 画像押下で、対象に遷移した状態のスライドが表示されます。
（ ファイル形式は jpeg・jpg・png・gif・pdf・txt のみ対応しています。 ）

やり取りをスムーズに行えるよう、非同期でのメッセージ自動更新を仕込みました。 より本格的なチャットアプリに近づけるため、メンション機能やスレッド機能等の追加を考えています。
